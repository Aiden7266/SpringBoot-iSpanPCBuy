中央處理器

解釋電腦指令以及處理電腦軟體中的資料。


1970年代前，中央處理器由多個獨立單元構成，後來發展出由積體電路製造的中央處理器， 這些高度收縮的元件就是所謂的微處理器。其中分出的中央處理器最為複雜的電路可以做成單一微小功能強大的單元， 也就是所謂的核心。中央處理器廣義上指一系列可以執行複雜的電腦程式的邏輯機器。

CPU散熱器

CPU 產生大量的熱量，高溫可能導致系統關機以保護重要元件，甚至可能導致永久性損壞。因此，為了保持 CPU 在低溫下運轉，風扇和散熱鰭片是必要配備。除了部分高階 CPU 外，大多數盒裝 CPU 都與對應的 CPU 散熱器一起搭售，可滿足散熱的基本需求。為了更好的系統效能，建議購買更好的 CPU 散熱器。

主機板

主要功能是提供一系列接合點，形成一個能整合處理器、記憶體、儲存裝置、顯示卡、音效卡、網卡和各種外部裝置的連接平台。主機板通常直接固定在機箱內壁，表面設計有可以直接插入擴充卡的插槽或可以連接其它內建裝置的線路連接。主機板上最重要的構成元件是晶片組。

記憶體

一種儲存資料的電子裝置。電腦記憶體可分為記憶體和外部記憶體，我們平常使用的程式，如Windows作業系統、打字軟體、遊戲軟體等，一般都是安裝在硬碟等外部記憶體上的，但必須把它們調入記憶體中執行，才能真正使用其功能，我們平時輸入一段文字，或玩一個遊戲，其實都是在記憶體中進行的，資料產生後不斷地由記憶體向外部記憶體進行讀寫。

顯示卡

簡稱顯卡，是個人電腦上以圖形處理器（GPU）為核心的擴充卡，用途是提供中央處理器以外的微處理器幫助計算圖像資訊，並將電腦系統所需要的顯示資訊進行轉換並提供逐行或隔行掃描訊號給顯示裝置，是連接顯示器和個人電腦主機板的重要元件，是「人機對話」的重要裝置之一。

硬碟

有時為了與後來出現的固態驅動器相區分而也稱「機械硬碟」或「傳統硬碟」，是電腦上使用堅硬的旋轉磁性碟片為基礎的非依電性記憶體，它在平整的磁性表面儲存和檢索數字資料，資料通過離磁性表面很近的磁頭由電磁流來改變極性的方式被寫入到磁碟上，資料可以通過碟片被讀取，原理是磁頭經過碟片的上方時碟片本身的磁場導致讀取線圈中電氣訊號改變。

電源供應器

電能轉換類的電源，主要功能為將標準交流電轉成低壓穩定的直流電，或直接將直流電轉成低壓穩定的直流電給某些特殊的器材使用。香港又稱火牛，是電牛的意思，意指電力供應者，源於早期變壓器造型似牛；用於電腦者，又俗稱電腦電源。

機殼

大部分可根據個人品味選擇，但有幾個需要牢記的因素。首先，機殼需要保持良好的風流，因此應該有許多風扇安裝點。同時要尋找前方可讓空氣進入的大開口，並加以過濾避免灰塵進入機殼。 其次，檢查您的機殼能夠容納的主機板大小，大多數主機板都是 ATX 大小，不適合較小的機殼。

週邊多媒體設備

週邊設備泛指主機以外的相關硬體設備，這些設備依其用途，可區分為輔助儲存設備（輔助記憶體）、輸入設備、及輸出設備等3大類
