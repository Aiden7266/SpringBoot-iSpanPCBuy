中央處理器(CPU)

CPU 會解釋電腦程式指令以及處理

電腦中的資料，讓電腦在開啟應用程式

或變更檔案時，完成所請求的任務。

CPU散熱器

CPU 會產生大量的熱量，高溫可能導致損壞，為了保護重要元件並保持 CPU 在低溫下運轉，建議購買好的 CPU 散熱器。

主機板

提供一系列接合點，形成一個能整合

處理器、記憶體、儲存裝置、顯示卡、

音效卡和各種外部裝置的連接平台。

記憶體

記憶體是系統的短期資料儲存區，存放電腦正在使用中的資訊，以便快速地存取。執行的程式越多，需要的記憶體就越多。

顯示卡

提供中央處理器以外的微處理器計算圖像

資訊，並將電腦系統所需要的顯示資訊

進行轉換並提供掃描給顯示裝置。

硬碟

硬碟是用來儲存電腦上的數位內容與資料的硬體。與電腦中的其他元件相互配合，在需要使用應用程式與檔案時呈現出來。

電源供應器

是電能轉換類的電源，將標準交流電轉成低壓穩定的直流電，或直接將直流電轉成低壓穩定的直流電給某些特殊器材使用。

機殼

機殼需要保持良好的風流，因此應該有許多風扇安裝點。需要可讓空氣進入的大開口，並加以過濾避免灰塵進入機殼。

週邊多媒體設備

週邊設備泛指主機以外的相關硬體設備，這些設備依其用途，可區分為輔助儲存設備、輸入設備、及輸出設備等3大類
